二、质点运动学

核心概念

  1. 位矢（位置矢量）：
    • 定义：从参考点指向质点的有向线段，r⃗ = xi⃗ + yj⃗ + zk⃗。
    • 大小：$r = \sqrt{x^2 + y^2 + z^2}$。
  2. 位移：
    • Δr⃗ = r⃗B − r⃗A，方向由初位置指向末位置。
  3. 速度与加速度：
    • 速度：$\vec{v} = \frac{d\vec{r}}{dt}$，加速度：$\vec{a} = \frac{d\vec{v}}{dt}$。
  4. 极坐标系下的运动：
    • 径向速度：$v_r = \frac{dr}{dt}$，横向速度：$v_\theta = r\frac{d\theta}{dt}$。
    • 加速度分量：$a_r = \frac{d^2r}{dt^2} - r\left(\frac{d\theta}{dt}\right)^2$，$a_\theta = r\frac{d^2\theta}{dt^2} + 2\frac{dr}{dt}\frac{d\theta}{dt}$。

典型例题

  • 例：已知r⃗ = 2ti⃗ + (2 − t2)j⃗，求轨迹方程及t = 0、t = 2s时的位矢。
    • 解：消去t得$y = 2 - \frac{x^2}{4}$；t = 0时r⃗ = 2j⃗，t = 2s时r⃗ = 4i⃗ − 2j⃗。

三、动量与动量守恒

核心公式

  1. 动量定义：
    • 单个质点：p⃗ = mv⃗。
    • 质点系总动量：p⃗ = ∑miv⃗i。
  2. 动量定理：
    • 质点：$\frac{d\vec{p}}{dt} = \vec{F}$。
    • 质点系：$\frac{d\vec{p}}{dt} = \vec{F}_{\text{外}}$（内力矢量和为零）。

重点概念

  • 惯性系与非惯性系：
    • 惯性系中牛顿定律成立，非惯性系需引入惯性力。
  • 保守力与势能：
    • 保守力做功与路径无关（如重力、弹力、万有引力），势能定义：A = −ΔU。

典型例题

  • 例：质量m的物体从弹簧自然长度下拉0.2m，求拉力做功。
    • 解：分两段积分，$A = \frac{1}{2}kx_0^2 + mg\Delta x = 3J$。

四、角动量与守恒定律

核心公式

  1. 角动量定义：
    • L⃗ = r⃗ × p⃗，方向由右手定则确定。
  2. 角动量定理：
    • $\frac{d\vec{L}}{dt} = \vec{r} \times \vec{F} = \vec{M}$（力矩）。
  3. 守恒条件：
    • 若M⃗外 = 0，则L⃗守恒。

典型例题

  • 例：求细杆绕端点转动的转动惯量。
    • 解：$J = \frac{1}{3}mL^2$（积分法）。

五、热学基础

核心概念

  1. 平衡态与状态方程：
    • 理想气体状态方程：pV = νRT。
  2. 麦克斯韦速率分布：
    • 分布函数：$f(v) = 4\pi \left(\frac{m}{2\pi kT}\right)^{3/2} v^2 e^{-mv^2/(2kT)}$。
    • 三种速率：最概然速率$v_p = \sqrt{\frac{2kT}{m}}$，平均速率$\bar{v} = \sqrt{\frac{8kT}{\pi m}}$，方均根速率$v_{\text{rms}} = \sqrt{\frac{3kT}{m}}$。

热力学第一定律

  • 公式：ΔU = Q − A（系统内能变化等于吸热减对外做功）。
  • 等温过程：$Q = A = \nu RT \ln\frac{V_2}{V_1}$。
  • 绝热过程：Q = 0，A = −ΔU。

六、循环过程与卡诺循环

核心公式

  1. 热机效率：
    • $\eta = \frac{A_{\text{净}}}{Q_{\text{吸}}} = 1 - \frac{Q_2}{Q_1}$。
  2. 卡诺循环效率：
    • $\eta_{\text{卡诺}} = 1 - \frac{T_2}{T_1}$（仅与高低温热源有关）。
  3. 致冷系数：
    • $w = \frac{Q_2}{A} = \frac{T_2}{T_1 - T_2}$。

典型例题

  • 例：斯特林循环效率计算。
    • 解：分过程计算吸放热，$\eta = 1 - \frac{Q_2}{Q_1} \approx 13.2\%$。

七、重点公式总结

  1. 运动学：
    • 极坐标速度：$\vec{v} = \frac{dr}{dt}\vec{e}_r + r\frac{d\theta}{dt}\vec{e}_\theta$。
  2. 动量与角动量：
    • 动量定理：$\vec{F} = \frac{d\vec{p}}{dt}$，角动量定理：$\vec{M} = \frac{d\vec{L}}{dt}$。
  3. 热力学：
    • 内能变化：ΔU = νCVΔT，等压过程：Q = νCpΔT。
  4. 循环过程：
    • 卡诺效率：$\eta = 1 - \frac{T_{\text{低}}}{T_{\text{高}}}$。

八、复习建议

  1. 概念梳理：重点理解守恒定律（动量、角动量、能量）的适用条件及物理意义。
  2. 公式应用：通过典型例题掌握公式的使用场景（如极坐标运动、热机效率计算）。
  3. 对比记忆：如保守力与非保守力、惯性系与非惯性系、三种速率的区别。
  4. 图解分析：结合p − V图理解循环过程，用速率分布图记忆麦克斯韦分布特点。

第二版（Deepseek详细版）

以下是更详细的复习资料整理，涵盖所有章节的核心内容、公式推导、典型例题及物理学思想：

普通物理B超详细复习手册

一、质点运动学（文档2、3）

1. 位矢与坐标系

  • 直角坐标系：
    • 位矢：r⃗ = xi⃗ + yj⃗ + zk⃗
    • 轨迹方程：消去时间参数t，如抛物线轨迹 $y = 2 - \frac{x^2}{4}$。
  • 极坐标系：
    • 位矢：r⃗ = re⃗r，其中e⃗r = cos θi⃗ + sin θj⃗，e⃗θ = −sin θi⃗ + cos θj⃗。
    • 速度分解： $$\vec{v} = \frac{dr}{dt}\vec{e}_r + r\frac{d\theta}{dt}\vec{e}_\theta$$
    • 加速度分解： $$\vec{a} = \left(\frac{d^2r}{dt^2} - r\omega^2\right)\vec{e}_r + \left(r\beta + 2\frac{dr}{dt}\omega\right)\vec{e}_\theta$$

2. 典型问题

  • 例题1：蚂蚁在旋转唱片上的轨迹
    • 极坐标方程：r = r0 + v0t，θ = θ0 + ω0t → 阿基米德螺线 $r = r_0 + \frac{v_0}{\omega_0}(\theta - \theta_0)$。
  • 例题2：三质点相互背离运动
    • 极坐标下速度分解，结合对称性求得轨迹为对数螺旋线。

二、动量与守恒定律（文档4、5）

1. 动量定理

  • 质点：$\frac{d\vec{p}}{dt} = \vec{F}$
  • 质点系：$\frac{d\vec{P}}{dt} = \sum \vec{F}_{\text{外}}$
  • 火箭推进：变质量问题，$\frac{dm}{dt} = -\frac{F}{v_{\text{喷}}}$。

2. 守恒条件

  • 动量守恒：∑F⃗外 = 0 → P⃗ = 常量。
  • 碰撞问题：
    • 完全弹性碰撞：动量守恒 + 动能守恒。
    • 非弹性碰撞：仅动量守恒。

3. 变力做功与功率

  • 功的计算：
    A = ∫F⃗ ⋅ dr⃗ = ∫x1x2Fxdx + ∫y1y2Fydy
  • 保守力场（重力、弹簧力、万有引力）：
    • 势能定义：U = −∫F⃗ ⋅ dr⃗
    • 如重力势能 U = mgh，弹性势能 $U = \frac{1}{2}kx^2$。

4. 典型例题

  • 例题：弹簧缓慢下拉问题
    • 分阶段计算变力做功：$A = \frac{1}{2}kx_0^2 + mg\Delta x = 3\,\text{J}$。

三、角动量与刚体动力学（文档6、7）

1. 角动量定理

  • 定义：L⃗ = r⃗ × p⃗，$\frac{d\vec{L}}{dt} = \vec{r} \times \vec{F} = \vec{M}$。
  • 守恒条件：M⃗外 = 0 → L⃗ = 常量。

2. 刚体定轴转动

  • 转动惯量：
    • 定义：J = ∑miri2
    • 常见刚体：
      • 细杆绕端点：$J = \frac{1}{3}mL^2$
      • 圆盘绕中心：$J = \frac{1}{2}mR^2$。
  • 转动定律：M⃗ = Jβ⃗。

3. 典型例题

  • 例题：摩擦力矩计算
    • 密度随半径变化的细杆：$M = -\frac{2}{3}\mu mgL$（积分法）。

四、热学与热力学（文档8、11）

1. 气体动理论

  • 平衡态与状态方程：
    • 理想气体：pV = νRT
    • 范德瓦尔斯方程：$\left(p + \frac{a}{V_m^2}\right)(V_m - b) = RT$（实际气体修正）。
  • 麦克斯韦速率分布：
    • 分布函数：$f(v) = 4\pi \left(\frac{m}{2\pi kT}\right)^{3/2} v^2 e^{-mv^2/(2kT)}$
    • 最概然速率 $v_p = \sqrt{\frac{2kT}{m}}$，平均速率 $\bar{v} = \sqrt{\frac{8kT}{\pi m}}$。

2. 热力学定律

  • 第一定律：ΔU = Q − A
    • 等温过程：$Q = A = \nu RT \ln\frac{V_2}{V_1}$
    • 绝热过程：Q = 0，A = −ΔU = −νCVΔT。
  • 第二定律：
    • 开尔文表述：不可能从单一热源吸热完全变成功。
    • 卡诺定理：$\eta_{\text{卡诺}} = 1 - \frac{T_2}{T_1}$。

3. 循环过程

  • 卡诺循环：
    • 效率：$\eta = 1 - \frac{T_2}{T_1}$
    • 逆卡诺循环（制冷机）：$w = \frac{T_2}{T_1 - T_2}$。
  • 斯特林循环：
    • 效率计算需分等温与等容过程，结合热机效率公式。

4. 典型例题

  • 例题：氧气瓶放气问题
    • 利用状态方程计算质量变化：$\Delta m = \frac{MV}{RT}(p_1 - p_2) = 0.015\,\text{kg}$。

五、物理学思想与研究方法

1. 模型与数学化

  • 质点模型：忽略形状，用位矢描述运动。
  • 理想气体模型：忽略分子体积与相互作用，简化热力学分析。

2. 守恒量的意义

  • 动量守恒 ↔︎ 空间平移对称性
  • 角动量守恒 ↔︎ 空间旋转对称性
  • 能量守恒 ↔︎ 时间平移对称性（诺特定理）。

3. 热力学与统计物理

  • 宏观与微观联系：温度 T 对应分子平均动能，压强 p 对应分子碰撞动量变化率。

六、公式速查表

章节 核心公式
运动学 $\vec{a} = \frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2}$
动量定理 $\vec{F} = \frac{d\vec{p}}{dt}$
角动量定理 $\vec{M} = \frac{d\vec{L}}{dt} = \vec{r} \times \vec{F}$
理想气体状态方程 pV = νRT
热机效率 $\eta = 1 - \frac{Q_2}{Q_1} = 1 - \frac{T_2}{T_1}$（卡诺循环）

七、复习策略

  1. 分模块突破：按运动学、动量、角动量、热学四大模块逐一梳理公式与典型例题。
  2. 推导训练：独立推导关键公式（如极坐标加速度、卡诺效率）。
  3. 真题演练：结合往年试题，分析高频考点（如守恒定律应用、循环过程计算）。
  4. 概念对比：列表对比动量与角动量、动能与内能等易混概念。
